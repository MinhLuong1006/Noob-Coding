
## I) len()  | Calculate the length of the string

```python
name = input()
result = len(name)
print(result)

#Input:Minh Luong
#Output:10
```

## II) find() | Find the first occurrence of an element in the string

```python
name = input()
result = name.find(L)
print(result)

#Input:Minh Luong
#Output:5
```

## III) rfind() | To find the last occurrence
```python

name = input()
result = name.rfind(L)
print(result)

#Input:Minh Luong
#Output:4
```
