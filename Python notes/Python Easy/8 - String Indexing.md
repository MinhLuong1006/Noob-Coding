## Indexing means accessing elements of a sequence using [] (indexing operator) [start : end : step]

```python
n = "Deutschlanduberalles"
print(n[0])


#Output:D
```

```python
n = "Deutschlanduberalles" #-1 starts at s
print(n[-3])


#Output:l
```

```python
n = "Deutschlanduberalles"
print(n[:4])


#Output:Deuts
```

```python
n = "Deutschlanduberalles"
print(n[15:])


#Output:alles
```

```python
n = "Deutschlanduberalles"
print(n[::2])


#Output:Dushadbrle
```
